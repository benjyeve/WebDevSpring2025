My Code

Tile Project (2025)

Created in Processing (Java) and ported over to p5.js, this project showcases custom mirror functions alongside for loops to create a set of repeating patterns. In the code, there are four unique patterns, a tile set, watermelon, smiley faces, and flowers. Created using Processing (Java), translated into p5.js.

Halloween Memorization (2020)

Having started this project right around Halloween time, I created a small little memory game where the goal is to match all of the tiles as fast as possible with as little moves as you can. Having been invested in several speedrunning communities at the time, I had my friends, who specialize in that exact field, compete to get the fastest times that they can. Created using p5.js.

Snake Clock (2020)

Obsessed with loops, I wanted to create a clock that not only tells the correct time, but incorporates the concept of a ouroboros snake as well. As a result, the snake clock was created; three snakes that represent hours, minutes, and seconds, respectively. Created using p5.js.
